# Flash-Cards-German2English
A popular and functional way of learning languages. Learn 3500 most popular German words in this fun application.
This application is for learning German language.  It's a popular flash card game. In the first 3 seconds a random German word is on display, after that appears a translation in English. If a user knew the word before the translated word appeared, he/she clicks the tick (right) button, otherwise clicks "X" (wrong) button. Words learned are no longer displayed to the user. There is also an option to switch from learning German to French.
